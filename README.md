# Emotion Detector

This is the final project for the IBM Python for AI & Development course.

## Project Description
An AI-based web application that detects emotions in text using the Watson NLP library. The application analyzes text input and returns emotion scores (anger, disgust, fear, joy, sadness) along with the dominant emotion.

## Technologies Used
- Python
- Flask
- Watson NLP Library
- Unit Testing (unittest)
- Pylint (Static Code Analysis)

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the Flask server: `python server.py`
3. Open browser at `http://localhost:5000`
